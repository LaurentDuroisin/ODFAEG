Bienvenue sur le site de ODFAEG.

ODFAEG est un framework de création d'applications qui se veut être à la fois flexible, performant et robuste!

Designer Templates

Le site a été créé à l'aide de git qui est une plateforme de développement très pratique surtout lorsque l'on veut développer à plusieurs, de ce fait chacun peut utiliser la version d'essai gratuitement et ainsi proposer des améliorations à l'avenir pour l'évolution du framework! Mais je n'ai plus autant de temps libre qu'avant donc il y va aussi de vous d'apporter votre aide.

Rather Drive Stick?

Si vous préférer ne pas utiliser toutes les fonctionnalités de ODFAEG ODFAEG . En complément vous pouvez utiliser d'autres frameworks grâce aux interfaces de ODFAEG, il est possible de n'utilise que SFML ou bien SDL ou encore QT pour des applications simples par exemple.

Authors and Contributors

Vous pouvez @mention un nom d'utilisateur git pour générer un lien vers votre profile. L'élément résultant <a> sera un lien vers le profile github du contribueur. Par exemple en 2007, Chris Wanstrath (@defunkt), PJ Hyett (@pjhyett), and Tom Preston-Werner (@mojombo) founded GitHub.
